# 使用截图说明

## 主界面

### 码表查询页面

码表查询页面提供完整的 GB2312 码表浏览和搜索功能：

- **区选择器**：快速切换 1-94 区
- **搜索框**：输入区位码进行搜索
- **字符网格**：以卡片形式展示所有汉字
- **字符详情**：点击可查看详细信息并复制

### 使用说明页面

使用说明页面包含：

- **区位码介绍**：什么是区位码及其编码规则
- **使用教程**：如何使用键盘输入汉字
- **添加键盘指南**：详细的键盘安装步骤
- **快捷按钮**：一键跳转到系统键盘设置
- **常用汉字参考**：区位码分布说明

## 键盘界面

### 键盘布局说明

```
┌─────────────────────────┐
│  区位码: 1601           │  ← 当前输入的区位码
│  啊                     │  ← 实时预览的汉字
└─────────────────────────┘
┌───────┬───────┬───────┐
│   1   │   2   │   3   │
├───────┼───────┼───────┤
│   4   │   5   │   6   │
├───────┼───────┼───────┤
│   7   │   8   │   9   │
└───────┴───────┴───────┘
┌─────┬───────┬─────────┬───────┐
│  0  │ 删除  │  空格   │ 换行  │
└─────┴───────┴─────────┴───────┘
```

### 功能说明

1. **顶部显示区**
   - 实时显示当前输入的区位码
   - 输入 4 位数字时显示对应汉字预览

2. **数字键盘区**
   - 大号数字按钮，方便输入
   - 标准 3x3 布局

3. **功能键区**
   - **0 按钮**：点击输入数字 0，**长按（0.5秒）切换键盘**
   - **删除**：优先删除输入缓冲区，然后删除文本
   - **空格**：输入空格字符
   - **换行**：输入换行符

## 使用流程示例

### 示例 1：输入"中国"

1. 切换到区位码键盘
2. 输入 `3021`
   - 显示：区位码: 3021 → 中
   - 自动插入"中"
3. 输入 `5448`
   - 显示：区位码: 5448 → 国
   - 自动插入"国"
4. 完成输入："中国"

### 示例 2：输入"你好"

1. 输入 `3694` → 你
2. 输入 `2345` → 好
3. 完成输入："你好"

### 示例 3：纠错

1. 输入 `16`
2. 输入错误，按"删除"键
3. 区位码变为 `1`
4. 继续输入正确的数字

## 码表查询使用

### 查询特定汉字

1. 打开主 App
2. 进入"码表查询"标签页
3. 在搜索框输入区位码（如 `1601`）
4. 查看搜索结果
5. 点击汉字查看详情
6. 可复制区位码或汉字

### 浏览某个区

1. 打开码表查询页面
2. 点击顶部的区选择器
3. 选择要浏览的区（如"第 16 区"）
4. 浏览该区所有汉字

### 查找高频字

常用高频字主要在第 16-55 区（一级汉字）：

- 第 16-19 区：a-d 开头的拼音
- 第 20-29 区：e-l 开头的拼音
- 第 30-39 区：m-q 开头的拼音
- 第 40-55 区：r-z 开头的拼音

## 设置说明

### 添加键盘

1. 打开主 App
2. 点击"使用说明"标签页
3. 阅读"如何添加键盘？"部分
4. 点击蓝色"打开键盘设置"按钮
5. 在打开的系统设置页面中：
   - 进入"键盘"设置
   - 点击"添加新键盘..."
   - 在第三方键盘中找到"区位码"
   - 点击添加

### 切换键盘

在任意输入框中：
- **长按（0.5秒）"0"按钮**
- 在弹出菜单中选择"区位码"

或使用系统地球图标（如果有其他键盘）。

## 技巧和建议

### 记忆常用字

建议记住一些高频字的区位码：

- `1684` 的
- `1822` 一
- `4263` 是
- `2934` 人
- `2976` 他

### 查找生僻字

1. 使用主 App 的码表查询
2. 按区浏览（第 56-87 区是二级汉字）
3. 记录需要的字的区位码

### 快速输入

- 熟练掌握数字键位置
- 记住常用字的区位码
- 利用实时预览确认输入

## 注意事项

1. **必须输入 4 位数字**：不足 4 位不会输出汉字
2. **区位范围**：区号和位号都是 01-94
3. **并非所有区位都有字**：某些位置为空
4. **长按切换**：**长按（0.5秒）"0"按钮**可切换键盘
5. **短按输入**：**点击"0"按钮**输入数字 0

## 故障排除

### 键盘不显示

- 确认已在系统设置中添加键盘
- 尝试重启设备
- 重新安装 App

### 无法输入汉字

- 检查是否输入了 4 位数字
- 确认区位码是否有效（01-94）
- 查看码表确认该位置是否有字

### 预览不显示

- 继续输入直到 4 位数字
- 确认区位码有效
